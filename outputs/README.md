# outputs/

Contains all files generated by running the pipeline. These are reproducible — delete and re-run `pipeline_v2.py` to regenerate.

| File | Description |
|---|---|
| `master_dataset.csv` | Cleaned, merged, and feature-engineered dataset (all 3 sub-tables joined) |
| `evaluation_report.txt` | Full model performance metrics, SHAP features, example prescriptions |
| `shap_summary_ph.png` | SHAP feature importance plot — pH model |
| `shap_summary_chlorine.png` | SHAP feature importance plot — Chlorine model |
| `shap_summary_turbidity.png` | SHAP feature importance plot — Turbidity model |
| `shap_summary_visit_timing.png` | SHAP feature importance plot — Visit timing model |
| `data_feasibility_report.docx` | Data column feasibility report (generated by `generate_report.py`) |
