# models/

Contains the trained XGBoost model files and supporting inference artifacts.
All files are generated by running `pipeline_v2.py` from the project root.

| File | Description |
|---|---|
| `xgb_ph.json` | Trained XGBoost model — predicts pH at next visit (MAE: 0.095, R²: 0.36) |
| `xgb_chlorine.json` | Trained XGBoost model — predicts free chlorine at next visit (MAE: 0.518, R²: 0.33) |
| `xgb_turbidity.json` | Trained XGBoost model — predicts turbidity at next visit (MAE: 0.116, R²: 0.43) |
| `xgb_visit_timing.json` | Trained XGBoost model — predicts deviation from seasonal visit baseline (MAE: 1.58 days) |
| `preprocessor.pkl` | Fitted sklearn ColumnTransformer — must be used with the above models for consistent feature encoding |
| `inference_config.json` | Feature lists, fill values, seasonal baselines, and regulatory thresholds — required for inference |

> **Regulatory basis:** Real Decreto 742/2013 (national) + Decreto 85/2018 (Comunitat Valenciana)
